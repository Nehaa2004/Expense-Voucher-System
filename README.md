# Expense-Voucher-System

## Project Overview

Expense Voucher System is a full-stack web application developed to manage employee expense vouchers digitally. The system is designed to simplify the process of creating, managing, and tracking expense requests.

The project is built using:

- Frontend: React.js
- Backend: Node.js + Express.js
- Database: MySQL
